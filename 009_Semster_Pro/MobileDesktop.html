Cross-Platform Native App with Xamarin.Forms

Shared Codebase using Xamarin.Forms

Developing cross-platform native apps with Xamarin.Forms allows you to share a significant portion of your codebase for Windows, macOS, iOS, and Android.

Example Xamarin.Forms Code:

           
   using Xamarin.Forms;
   
   namespace MyApp
   {
      public partial class App : Application
      {
         public App()
         {
            InitializeComponent();
            MainPage = new NavigationPage(new MainPage());
         }
      }
   
      public class MainPage : ContentPage
      {
         public MainPage()
         {
            Title = "Xamarin.Forms App";
   
            Content = new StackLayout
            {
               Children = {
                  new Label { Text = "Hello, Xamarin.Forms!" },
                  new Button { Text = "Click Me!", Command = new Command(OnButtonClicked) }
               }
            };
         }
   
         private void OnButtonClicked()
         {
            DisplayAlert("Button Clicked", "You clicked the button!", "OK");
         }
      }
   
      // Add more pages and shared code as needed
   }

This code defines a simple Xamarin.Forms application with a main page containing a label and a button. The button click event shows an alert.

Conclusion

You've set up a Xamarin.Forms project with shared code for building native apps on Windows, macOS, iOS, and Android. This example demonstrates a simple UI with a button click event. Further, you can create platform-specific UI as needed. Happy coding!
